Для ініціалізації депозиторія робимо git init
Щоб зрозуміти статус репозиторію на даний моент - git status
Щоб додати файл - git add<імя файлу>
Підключитися до віддаленого репозиторію - git remote add origin < ссілка>
Підкачати зміни файлу до віддаленого репозиторію - git push origin master
Подивитися коміти - git log
Щоб створити новий бранч - git checkout -b "назва бранча"
